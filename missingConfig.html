ERROR: arloweb.ini not found:

Please modify arloweb.ini.example to set your folder location and user id,
then rename the file to arloweb.ini

You will also need entries in /etc/sudoers to allow whatever user your web server runs under to run startROS.sh and stopROS.sh.
I use XAMPP, which runs as daemon,
so I have this line in /etc/sudoers:
daemon ALL = (chrisl8) NOPASSWD: /home/chrisl8/arloweb/startROS.sh, /home/chrisl8/arloweb/stopROS.sh

Try again?
